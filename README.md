
# 🍥 火影忍者·忍术结印系统

## ✨ 功能特性

### 🔍 智能搜索
- 支持忍术名称、罗马音、使用者多维度搜索
- 实时联想建议，显示属性图标和忍术信息
- 键盘导航（↑↓ 选择，Enter 确认，Esc 关闭）

### 🏷️ 多维度筛选
- **属性筛选**：火遁 🔥 / 水遁 💧 / 土遁 🪨 / 雷遁 ⚡ / 风遁 🌪️ / 其他 🌀
- **结印筛选**：🤙 一键过滤有结印序列的忍术

### 🎭 结印动画系统
- **12种标准结印**完整收录（子丑寅卯辰巳午未申酉戌亥）
- 逐印高亮播放动画，可调节速度
- 高亮区常驻显示，播放完成后保持最终状态
- 支持重新播放

### 🔊 属性音效系统
- **结印音效**：模拟手指结合 + 查克拉波动的短促音效
- **释放音效**：6种属性独立音效设计
  - 🔥 火遁：噼啪燃烧 + 爆燃冲击
  - 💧 水遁：水流冲刷 + 浪涛拍击
  - 🪨 土遁：岩石碎裂 + 沉闷震动
  - ⚡ 雷遁：电弧放电 + 嗡鸣持续
  - 🌪️ 风遁：呼啸气流 + 风声消散
  - 🌀 无属性：纯查克拉爆发脉冲
- 基于 Web Audio API 实时合成，无需加载音频文件

### 📖 忍术百科
- 146+ 忍术完整数据库
- 每个忍术包含：日文名、罗马音、使用者、等级、属性、结印序列、详细描述
- 覆盖原作主要角色和经典忍术

## 🚀 快速开始


## 🛠️ 技术实现

### 结印动画

逐印播放，每印触发高亮和音效，完成后高亮区常驻显示。

### 属性音效合成

基于 Web Audio API 多层音频实时合成：

```
第1层  查克拉聚集   sine    80→300Hz     蓄力感
第2层  能量爆发     属性波形  属性频率     爆发冲击
第3层  冲击波噪声   noise   有色噪声     爆炸质感
第4层  低频震动     sine    50→25Hz      地面感
第5层  余韵泛音     sine    高频衰减     回响消散
第6层  属性特色     各异    各异         属性辨识
```

### 搜索系统

输入关键词 → 多维度匹配（名称/罗马音/使用者）→ 实时渲染建议列表 → 键盘导航或鼠标点击选择 → 加载忍术 → 渲染卡片和结印时间线。

## 🔧 自定义与扩展

### 添加新忍术

在 script.js 的 JUTSU_DATABASE 中添加：

```javascript
'你的忍术名': {
  romaji: 'Romaji Name',
  user: '使用者A / 使用者B',
  rank: 'S',                    // E/D/C/B/A/S
  element: 'fire',              // fire/water/earth/lightning/wind/none
  seals: ['子', '丑', '寅'],    // 空数组表示无需结印
  description: '忍术的详细描述',
},
```

### 可用结印

```javascript
['子', '丑', '寅', '卯', '辰', '巳', '午', '未', '申', '酉', '戌', '亥']
```

```
fire       🔥 火遁

earth      🪨 土遁
lightning  ⚡ 雷遁
wind       🌪️ 风遁
none       🌀 无属性
```
